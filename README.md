# Magnet-Hop
Minimalist mobile arcade game featuring magnetic rail flipping mechanics.


## Features
- One tap gravity flip
- Magnetic rail snapping
- Endless obstacle runner
- Mobile-first design

## Engine
Unity 2D (C#)

## Target Platforms
Android / iOS
